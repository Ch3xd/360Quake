# 360Quake

360 Quake空间搜索引擎api调用

## Installation

```bash
git clone https://github.com/Ch3xd/360Quake.git
cd 360Quake
pip3 install -r requirements.txt
```



## Usage

```bash
python3 main.py -k xx-xxx-xxx -q query_str
```

![image-20221215162357487](img/1.png)

运行完后会在当前目录下生成三个文件：

`url.txt  存放URL链接`

`domain.txt 存放域名信息`

`ips.txt 存放IP链接`
